# ADIVINHAÇÃO DE NÚMEROS

O jogo foi criado como forma de treinamento, para moldar meus conhecimentos em python, por isso foi totalmente criado na linguagem python e ainda não foi muito extendido. 

## 💻 Sobre o projeto

Um game de adivinhação, onde você escolherá um número e se for o mesmo que a máquina optou, ganhará a rodada, se não, dependendo do tipo de dificuldade, terá mais chances ou menos e ao final, após ganhar, ele mostrará quantas tentativas o jogador precisou para adivinhar.
